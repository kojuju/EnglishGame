# 英语分级闯关小游戏

一个用纯 HTML、CSS、JavaScript 写的单机网页小游戏，支持按英语水平切换词库，并提供“词义选择 / 听写”两种练习模式，目标是让不同阶段的学习者都能用不过纲的单词练词义识别和拼写。

## 直接运行

打开 `index.html` 即可开始试玩。

> 注意：**听写模式**在 `file://` 直接打开时有时也能运行，但不稳定；更推荐通过本地服务器访问（例如 `http://localhost`），以避免浏览器拦截语音能力。

### 推荐的一键启动方式

直接双击 `start-game.cmd`，它会：

1. 在本地启动一个 `http://localhost` 静态服务
2. 自动打开浏览器进入游戏
3. 保持当前窗口运行，关闭窗口或按 `Ctrl+C` 可停止服务

## 当前版本功能

- 首页 + 闯关页 + 结算页
- 练习模式菜单：`词义选择 / 听写`
- 设置弹窗，可切换 `初学者 / 小学 / 中学 / 大学 / CET-4 / CET-6`
- 词义选择：每局 10 题，带倒计时、生命值、连击、分数
- 听写：每局 20 词，每词自动播放 3 遍，间隔 5 秒后继续播放
- 结果页支持“本局错题再战”，两种模式都可直接重练本局错题
- 分级词库，6 个级别均扩充到 1000 词，默认 CET-4
- 题面展示音标
- 局后错词回顾展示音标、多词性、多义项
- 最高分 / 最佳正确数、最近正确率、错词记录按“玩法 + 级别”分别本地保存

## 文件说明

- `index.html`：页面结构
- `generated-word-bank.js`：按级别补齐到 1000 词的扩展词库
- `styles.css`：样式与响应式布局
- `words.js`：词库数据
- `app.js`：游戏逻辑和本地存储
- `start-game.cmd` / `start-game.ps1`：一键启动本地服务并打开浏览器
- `requirements\`：需求与设计文档目录（包含 `01-requirements.md` ~ `07-wrong-answer-rematch-analysis.md`）
